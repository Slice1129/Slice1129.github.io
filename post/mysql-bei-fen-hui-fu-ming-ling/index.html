MySQL 备份恢复命令

2019-06-28
  • 备份指定的数据库并压缩。
 ./mysqldump -h 127.0.0.1 -u root -p kdej(需要备份的数据库名) | gzip > /usr/local/kdej.sql.gz

按回车后，会提示输入密码，然后开始备份。通过网络传输至新服务器上，然后开始恢复。

  • 恢复数据库
gunzip < /usr/local/kdej.sql.gz | mysql -h 127.0.0.1 -u root -p kdej(库名)
